Metodología Aplicada

1. Evaluación de Necesidades

Se identificaron los requerimientos específicos de gestión financiera ("pain points") de Las Araucarias para personalizar el sistema según sus flujos de caja reales.

2. Selección de Tecnología

Se optó por el ecosistema AppSheet + Google Sheets por su bajo costo operativo (No-Code), facilidad de mantenimiento y capacidades de integración en la nube.

3. Diseño de Interfaz & UX

Desarrollo de una interfaz intuitiva adaptada a usuarios no técnicos, simplificando la entrada de datos transaccionales para reducir el error humano.

Análisis Técnico

Para la automatización de ciertos cálculos, se utilizaron expresiones lógicas dentro de AppSheet similares a Excel/SQL. Ejemplo de lógica de validación:

AppSheet Expression 

IF([Estado] = "Pendiente", 
   NOW() + "7", 
   [Fecha_Pago]
)
// Valida fechas de vencimiento automático
