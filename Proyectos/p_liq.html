Excel & VBA Automation

Sistema Automatizado de Remuneraciones y Análisis de Nómina

Solución integral de gestión de nómina que automatiza el ciclo completo de remuneraciones. El sistema utiliza Power Query para extraer indicadores económicos en tiempo real (Web Scraping) y un motor de cálculo en VBA para procesar sueldos, impuestos y leyes sociales, garantizando precisión contable y una experiencia de usuario (UX) moderna.

Metodología Aplicada

1. Ingeniería de Datos (ETL)

Implementación de conexiones mediante Power Query para la extracción automatizada de indicadores económicos (UF, UTM) desde portales oficiales (SII/Previred).

2. Arquitectura de Datos

Diseño relacional para la gestión de empleados y desarrollo de un calendario lógico para el control de asistencia y semana corrida.

3. Motor de Cálculo (VBA)

Programación de macros y lógica financiera para determinar gratificaciones, impuestos y leyes sociales automáticamente.

4. Visualización y Reporting

Consolidación de la información en un Dashboard de KPIs y generación automatizada de documentos PDF listos para firmar.

Análisis Técnico

El núcleo del proyecto transforma a Excel en una aplicación de escritorio robusta, combinando la flexibilidad de las hojas de cálculo con la potencia de la programación VBA.

Automatización con Macros (VBA)

Se desarrollaron scripts para gestionar el flujo de trabajo: navegación entre módulos, guardado de históricos en bases de datos y exportación de reportes a PDF. Esto reduce procesos de horas a segundos.

Ingeniería de Datos y Web Scraping
  • - Power Query: Conexión directa a fuentes web (Previred/SII) para mantener los parámetros legales (UF, UTM, Tramos) siempre actualizados.
  • - Modelado: Estructura de bases de datos relacionales (Empleados, Histórico Calendario, Histórico Liquidaciones) dentro del mismo libro.

Validación y Lógica Compleja

Uso de fórmulas matriciales y validación de datos para prevenir errores humanos en la entrada de información (días trabajados, horas extras), asegurando la integridad contable.

Principales Insights (Valor Agregado)

Automatización Extrema

Eliminación de la carga manual de datos económicos mediante Web Scraping, asegurando cálculos siempre alineados a la normativa vigente sin intervención humana.

UI/UX Orientada al Usuario

Navegación hipervinculada estilo web que optimiza la usabilidad y reduce drásticamente la curva de aprendizaje para el operador del sistema.

Control de Gestión

Transformación de datos operativos en indicadores financieros (KPIs) en tiempo real, permitiendo monitorear costos de empresa y dotación al instante.

Integridad y Trazabilidad

Historial completo de registros en bases de datos internas que permite auditorías rápidas y reportabilidad histórica precisa de cada peso pagado.
