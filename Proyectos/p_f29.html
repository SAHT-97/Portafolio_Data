"Auditoría Tributaria: Procesador Automático de Formularios 29 en PDF"

El objetivo de este proyecto es crear una aplicación en Python (Streamlit) que permita procesar automáticamente Formularios 29 en formato PDF (versión compacta), extrayendo por códigos oficiales del formulario, estructurándolos tabularmente y calculando reajuste de remanente y actualización de PPM automáticamente.

Python, Pandas, Streamlit, PDF Parsing

Metodología y Funcionalidades Principales

1. Carga de Archivos Múltiples

El sistema permite al usuario subir una carpeta completa o seleccionar múltiples archivos PDF, correspondientes a Formularios 29 de distintos períodos tributarios.

2. Lectura y Extracción de Datos Robusta

El análisis de cada PDF extrae automáticamente los valores relevantes basándose en los códigos oficiales del formulario F29 (no en posiciones fijas), asegurando robustez ante las variaciones de formato dentro de los PDFs.

3. Estructuración Tabular de la Información

Los datos extraídos se organizan de manera limpia en un DataFrame de pandas. Cada fila representa un formulario o período distinto y cada columna expone el valor de un código relevante procesado.

4. Cálculos Automáticos Tributarios

El sistema calcula inteligentemente el PPM (Pago Provisional Mensual) actualizado y ejecuta el reajuste del crédito fiscal aplicando las reglas tributarias pertinentes, contrastando el valor real declarado vs el cálculo sistémico.

5. Visualización de Resultados y Filtros

Las tablas finales se disponen a través de una interfaz clara e intuitiva en Streamlit. Es posible visualizar valores originales, calculados con la normativa, y aplicar filtros de periodos para revisión cruzada.
