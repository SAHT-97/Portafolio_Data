Sistema de Conciliación Bancaria a Medida

El objetivo central de este desarrollo fue diseñar e implementar un sistema de conciliación bancaria exclusivo para una oficina contable. Este sistema agrupa toda la información de pagos, cobros, cuentas y bancos en un único archivo de aplicación gestionado por Google Apps Script, de manera que el proceso diario se vuelve una tarea rápida, visual, precisa y libre de fricciones.

Metodología y Flujo de Uso

1. Selección del Movimiento

Búsqueda inteligente en la "Vista" principal, eligiendo movimientos bancarios pendientes para visualizar el monto disponible, banco y concepto.

2. Vinculación con Cobros

Búsqueda de clientes por ID o nombre, seleccionando deudas pendientes en el sistema para cruzar con el movimiento bancario.

3. Asignación de Pagos e Impuestos

Inclusión de pagos y obligaciones del cliente (como impuestos) en el balace del movimiento del banco de manera intuitiva.

4. Ajustes y Conciliación Final

Ingreso de comisiones o conceptos adicionales, con un cálculo matemático en tiempo real que permite conciliar de manera exacta y guardando la información estructurada.

Análisis Técnico

Para dotar al sistema de esta eficiencia y poder de procesamiento, se emplearon las siguientes estrategias técnicas:

ETL (Extracción, Transformación y Carga)

Se empleó Google Colab y Python (Pandas) para recuperar, limpiar profundamente y normalizar la información dispersa en múltiples archivos Excel.

ETL.py 
from os import pathconf_names
import pandas as pd

# ── CARGA DE DATOS ──────────────────────────────────────────────────────────
clientes = pd.read_excel("/content/drive/MyDrive/SVS/Libro1.xlsx")

pathconf_c = "/content/drive/MyDrive/SVS/5.- PLANILLA DE COBROS 2026.xlsx"
cobros = pd.read_excel(pathconf_c, skiprows=2)

# ── LIMPIEZA INICIAL ─────────────────────────────────────────────────────────
# Se encadena todo en un solo bloque para evitar reasignaciones innecesarias
cobros_work = (
    cobros
    .iloc[:-14, 1:27]          # Filas sin totales + columnas útiles
    .copy()
)
cobros_work.columns = cobros_work.columns.str.strip()

# ── COLUMNAS A ELIMINAR ──────────────────────────────────────────────────────
COLS_A_BORRAR = [
    'Columna1', 'Ventas', 'Honorarios', 'Tasa de PPM', 'Software',
    'Imp. Pagadas', 'IVA Pagado', 'TGR Pagado', 'Otros Pagado',
    'Renta Pagada', 'Extra Pagado', 'Pdte. Pagado', 'Mens Pagado'
]
cobros_work.drop(columns=COLS_A_BORRAR, errors='ignore', inplace=True)

# ── MELT (PIVOTE LARGO) ──────────────────────────────────────────────────────
# Se usa la primera columna como identificador (presumiblemente 'Rut' u otro ID)
id_col = cobros_work.columns[0]  # Nombre explícito, más legible que una lista

cobros_melted = cobros_work.melt(
    id_vars=id_col,
    var_name="Cuentas",
    value_name="Monto"
)

# ── LIMPIEZA DE MONTO ────────────────────────────────────────────────────────
cobros_melted['Monto'] = (
    pd.to_numeric(cobros_melted['Monto'], errors='coerce')
    .fillna(0)
)

# Filtrar montos cero (se hace ANTES del merge para reducir el tamaño del df)
cobros_melted = cobros_melted[cobros_melted['Monto'] != 0].copy()

# ── MERGE CON CLIENTES ───────────────────────────────────────────────────────
# CORRECCIÓN: el merge se hace sobre la columna id_col, no hardcodeado en 'Rut'
# Ajusta left_on si tu columna identificadora tiene otro nombre
cobros_id = cobros_melted.merge(
    clientes[['Rut', 'ID', 'Cliente']],
    left_on=id_col,
    right_on='Rut',
    how='left'
)

# Orden final de columnas
cobros_id = cobros_id[['Cuentas', 'ID','Rut', 'Cliente', 'Monto']]

# ── RESUMEN ──────────────────────────────────────────────────────────────────
pd.options.display.float_format = '${:,.0f}'.format

resumen = (
    cobros_melted
    .groupby('Cuentas')['Monto']
    .agg(N_Movimientos='count', Monto_Total='sum')
    .reset_index()
    .rename(columns={'Cuentas': 'Tipo de Cuenta'})
)

print("── RESUMEN POR CUENTA ──")
display(resumen)

banco_melted = pd.melt(
    banco,
    id_vars= cf_banco,
    var_name= "Banco",
    value_name= "Monto",
)

nuevo_orden = ['Banco'] + cf_banco + ['Monto']
f_banco  = banco_melted[nuevo_orden]
f_banco.dropna(subset=['Monto'], inplace=True)



# 1. Aseguramos que la columna Monto sea numérica
# 'errors='coerce'' transformará cualquier texto sucio (como "$", "Total", "-") en NaN (vacío)
f_banco['Monto'] = pd.to_numeric(f_banco['Monto'], errors='coerce')

# 2. (Opcional) Si quieres convertir los vacíos a 0 para que no afecten la suma
f_banco['Monto'] = f_banco['Monto'].fillna(0)








# 3. AHORA sí ejecuta tu agrupación
resumen = f_banco.groupby('Banco')['Monto'].agg(['count', 'sum']).reset_index()

# Verificamos

# 3. Formato opcional (para que se lea mejor en pantalla)
# Esto no cambia los datos, solo cómo se imprimen
pd.options.display.float_format = '${:,.0f}'.format
