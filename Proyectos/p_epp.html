"Sistema de Optimización y Control de Entrega de EPP"

Este proyecto nació de una necesidad real: transformar un registro anual estático en una herramienta dinámica de gestión operativa. Desarrollé un sistema integral en Excel que automatiza la asignación de Elementos de Protección Personal (EPP), permitiendo el seguimiento en tiempo real de costos por trabajador y estados de entrega. Mediante el uso de macros y dashboards interactivos, logré reducir los tiempos de respuesta logística y eliminar la opacidad en el gasto departamental.

Metodología Aplicada

1. Diagnóstico

Identificación de cuellos de botella en el manejo de un Excel plano de pedidos anuales.

2. Normalización de Datos

Limpieza y estructuración de bases de datos de trabajadores y catálogos de productos (EPP).

3. Desarrollo de Lógica

Creación de motores de búsqueda y formularios de registro automatizados..

4. Diseño de Interfaz

Construcción de un Dashboard interactivo para la toma de decisiones gerenciales

Análisis Técnico

Este proyecto, utilicé técnicas intermedias de Excel para garantizar la integridad y agilidad de los datos:

Automatización con VBA (Macros)

Creación de un script para el registro automático de entregas, moviendo datos del formulario a la base histórica con un solo clic.

Funciones de Búsqueda y Referencia

Implementación de BUSCARV y validación de datos para listas dinámicas que extraen información en tiempo real de múltiples tablas.

Modelado de Datos

Uso de Tablas y Gráficos Dinámicos para procesar grandes volúmenes de información de manera eficiente.

Interfaz de Usuario (UI)

Segmentación de datos (Slicers) para crear una experiencia de navegación intuitiva y filtros avanzados por nombre para la gestión de personal.

Principales Insights (Valor Agregado)

Control Financiero Preciso

Implementación de un rastreo de costos por trabajador y departamento, permitiendo una visión clara del presupuesto ejecutado.

Eficiencia Operativa

Reducción del tiempo de búsqueda y entrega en bodega mediante la automatización de consultas de tallas y existencias.

Trazabilidad Total

Transición de un pedido anual estático a un sistema dinámico con estados de entrega en tiempo real ("Entregado" vs. "Pendiente").

Visualización de Resultados

Ver Detalle

Este tablero principal es el núcleo de la toma de decisiones. Permite visualizar KPIs críticos como el porcentaje de EPP entregado vs. pendiente, el valor monetario por categoría y el consumo por turno. Gracias a la segmentación de datos, cualquier reclutador o jefe de área puede filtrar por trabajador o producto en segundos.

El módulo de registro utiliza listas dinámicas que vinculan la base de trabajadores con el catálogo de precios de EPP. La magia ocurre detrás: una macro de VBA procesa la información y la archiva automáticamente, asegurando que no haya errores humanos en la entrada de datos y manteniendo el historial siempre actualizado.
